SpectatorPlus

This project contains the source of a plugin for Bukkit (v1.0-v2.0), and Spigot (v2.0+).

    Commands & Permissions

    [] parameters are optional; <> parameters are required.
    Some optional parameters are required when using the console.

    Command Permission Defaults to Description
    /spectate [help] or /spec [help] none everyone Displays a list of all SpectatorPlus commands.
    /spec on [player] spectate.use (self) and spectate.use.others (other players) op Enables spectate mode [for the specified player]
    /spec off [player] spectate.use (self) and spectate.use.others (other players) op Disables spectate mode [for the specified player]
    none spectate.admin op Grants all sub-permissions:
    • spectate.admin.lobby,
    • spectate.admin.mode,
    • spectate.admin.arena,
    • spectate.admin.reload,
    • spectate.admin.hide,
    • spectate.admin.hide.see
    • spectate.admin.say,
    • spectate.admin.config, and
    • spectate.use.others.
